# История изменений

## 1.7.2 - 2026-07-23

- Проект переименован в VoxCortex; прежние пользовательские данные и сетевое имя сервера
  автоматически поддерживаются при переходе.
- Конфигурация, история и журналы перенесены в `%LOCALAPPDATA%\VoxCortex`; папку моделей по-прежнему
  можно выбрать в настройках.
- Исправлено переподключение устройства после изменения адреса ПК или замены роутера.
- Добавлена добровольная поддержка проекта через ЮKassa: ссылка и QR-код доступны на GitHub, в
  главном окне, настройках и меню трея.
- Релизные ZIP проверяются на отсутствие моделей и пользовательских данных, получают контрольные
  суммы и GitHub artifact attestations, а GitHub Release создаётся как черновик для ручной проверки.
- Возвращены безопасные тесты миграции, конфигурации, истории, обновления прошивки и состава
  релизного архива; версии GitHub Actions и PlatformIO закреплены на проверенных значениях.

## 1.7.1 - 2026-07-19

- Проверка версии и USB-прошивка встроены в главное Windows-приложение: доступны список CH9102,
  текущая и доступная версии, журнал записи и проверка результата после перезагрузки.
- Обычное обновление записывает только приложение и сохраняет Wi-Fi; полная очистка вынесена в
  отдельный режим `--factory` с явным предупреждением.
- Версия прошивки задаётся в одном JSON-файле и автоматически встраивается вместе с Git commit.
- GitHub Actions проверяет сервер и прошивку, а тег версии создаёт ZIP главного приложения со
  встроенным проверенным пакетом прошивки и отдельный пакет бинарных файлов для разработчиков.
- Из `platformio.ini` удалён локальный `COM11`; порт теперь определяется во время обновления.
- Известные галлюцинации Whisper отбрасываются, а AUTO требует более устойчивого звука для запуска.
- В главном окне показывается имя активной модели без пути к её файлам.

## 1.7.0 - 2026-07-19

- Модели faster-whisper по умолчанию загружаются в папку `models` рядом с конфигурацией и EXE, поэтому
  весь дистрибутив можно перенести на другой ПК вместе с моделями.
- В настройках можно выбрать произвольную папку моделей, открыть её в Проводнике и удалить
  неиспользуемую загруженную модель с подтверждением.
- Активную модель нельзя удалить: сначала нужно выбрать и применить другую, что исключает повреждение
  текущего распознавания.
- Сценарий пересборки EXE сохраняет `models`, `config.yaml`, `history.json` и `logs` вместо удаления
  пользовательских данных вместе со старой папкой приложения.
- Прогресс загрузки рассчитывается по суммарному объёму всех файлов модели: маленькие служебные файлы
  больше не переводят индикатор преждевременно на 99% перед долгой загрузкой `model.bin`.

## 1.6.1 - 2026-07-19

- Распознавание зафиксировано на русском языке, чтобы короткая тихая речь не переключала Whisper на
  случайный иностранный язык.
- Для коротких фраз усилено отбрасывание неуверенной тишины, а контекст предыдущей фразы больше не
  переносится в следующую.
- Переход `AUTO LISTEN` ↔ `AUTO SPEECH` обновляет только заголовок и таймер без очистки экрана и
  сброса спектра.

## 1.6.0 - 2026-07-19

- Двойное короткое нажатие кнопки M5 включает и выключает автоматическую диктовку без удерживания.
- AUTO использует адаптивный порог шума, 640 мс предзаписи, подтверждение начала речи и завершение
  фразы после примерно 1,2 секунды тишины.
- Одиночное короткое нажатие M5 в AUTO вручную завершает текущую фразу, C отменяет её, не выходя из
  автоматического режима.
- Сервер принимает следующую фразу во время расшифровки предыдущей и обрабатывает очередь строго по
  порядку; статусы содержат номер соответствующей записи.
- Обычный Push-to-Talk сохранён и начинается после распознавания удержания кнопки M5.

- Применение настроек больше не перезапускает сервер и не затрагивает содержимое главного окна.
- Все основные настройки собраны в одном окне; убраны имя устройства и технические поля языка,
  вычислителя, типа вычислений и beam size.
- Добавлен выбор мультиязычной модели Whisper, предупреждение о загрузке и индикатор её состояния.
- Список моделей сразу показывает, какие варианты уже загружены; для выбранной модели выводится
  точный путь хранения, а скачивание показывает процент и переданный объём в отдельном временном окне.
- Технический выбор CPU/CUDA удалён: распознавание всегда работает в совместимом режиме CPU/int8.
- Tray-меню вместо порта показывает подключённое устройство; из него можно
  переключить модель, а технический переход к JSON-состоянию сервера удалён.
- Выбранная модель подготавливается в фоне при каждом запуске и сразу после смены, чтобы первое
  распознавание не ожидало загрузки модели в память.
- Язык речи определяется автоматически.
- Последние 250 распознанных фраз сохраняются в `history.json` и восстанавливаются после запуска.

## 1.5.0 - 2026-07-19

- Окно истории стало главным окном приложения со статусом M5Stick, редактором текста и выбором
  действия вывода.
- Ручная вставка возвращает фокус в последнее активное внешнее окно; фразы из истории загружаются в
  редактор щелчком.
- Добавлено окно настроек устройства, faster-whisper и вставки с атомарным сохранением `config.yaml`.
- Приложение синхронизирует имя, действие, максимальную длительность и звуковые сигналы с новой
  прошивкой, сохраняя совместимость со старой прошивкой протокола v1.
- В карточке устройства отображаются батарея, зарядка, уровень Wi-Fi и версия прошивки.
- Выбор действия перенесён в приложение; короткая B на устройстве теперь напоминает изменить режим
  на ПК.

## 1.4.0 - 2026-07-15

- Дублирующие профиль приложения и режим отправки заменены одним действием вывода.
- Доступны четыре однозначных действия: только копирование, вставка, вставка с Enter и вставка с
  Ctrl+Enter.
- Короткое нажатие B циклически меняет действие; длительное нажатие освобождено для будущей функции.
- Прошивка автоматически переносит сохранённые настройки 1.3.0 в новое действие без очистки NVS.
- Протокол переходной версии передаёт новое действие и совместимые старые поля, поэтому сервер и
  прошивку можно обновлять в любом порядке.
- Настройка отдельных профилей удалена из пользовательского шаблона конфигурации сервера.

## 1.3.0 - 2026-07-15

- Линейная шкала микрофона заменена спектром из 14 частотных полос от 100 Гц до 7 кГц.
- Спектр обновляется без полной очистки экрана и использует быстрый подъём с плавным затуханием.
- Цвет отображает уровень сигнала: зелёный — нормальный, жёлтый — высокий, красный — клиппинг.
- Клиппинг определяется по насыщенным PCM-отсчётам; лишние текстовые индикаторы уровня убраны.
- Начальный номер записи рандомизируется после загрузки, исключая `Duplicate sequence` после прошивки.

## 1.2.1 - 2026-07-15

- Запуск микрофона теперь выполняется только после подтверждения записи сервером.
- Устранено повторное выключение I2S-динамика при каждом нажатии A.
- Запоздалый аудиоблок больше не разрывает WebSocket-соединение.
- Тишина возвращает устройство в готовое состояние без аварийного `ERROR`.
- Экран записи получил отдельную разметку без наложения строк и полной частой перерисовки.
- Убрана двойная надпись `Hold A to speak`, добавлен вывод переходов состояния в монитор порта.
- При достижении максимальной длительности отправляются оставшиеся аудиоблоки.

## 1.2.0 - 2026-07-15

- Добавлено окно истории распознавания с временными метками и кликабельными фразами.
- Щелчок по фразе повторно копирует её в буфер обмена.
- Окно открывается при запуске и сворачивается в область уведомлений; двойной щелчок возвращает его.
- Автоматическая вставка переведена с PyAutoGUI на нативный Windows `SendInput`.
- Распознанный текст больше не записывается в диагностический `server.log`.

## 1.1.0 - 2026-07-15

- Добавлено Windows tray-приложение сервера с отдельным EXE и значком состояния.
- Распознанные фразы записываются в журнал как структурированные события `recognition`.
- Журнал теперь состоит из корректных UTF-8 JSON-строк, включая кавычки и многострочный текст.
- Добавлены скрипты сборки EXE и явного включения автозапуска.
- Сборка выравнивает версии библиотек Microsoft C++ Runtime для стабильной загрузки CTranslate2.
- Зафиксировано проверенное состояние устройства и сервера.

## 1.0.0 - 2026-07-14

- Первая версия прошивки, локального сервера, симулятора, тестов, Windows-скриптов и документации.
